# Hypotheses - Project 4

**Title**: The impact of leader workaholism on female working students' reproductive health

**Team**: Quin & Michelle

## Model

```
Leader Workaholism (Cognitive) → ICT → Heavy Menstrual Bleeding
```

## Hypotheses

**H1**: Leader workaholism (cognitive) is positively related to employees' after-hours ICT use.

**H2**: After-hours ICT use is positively related to heavy menstrual bleeding.

## Analysis Plan

1. Descriptive statistics and correlations
2. Reliability analysis (Cronbach's alpha for MWS_Cognitive)
3. Regression analyses (H1, H2)
